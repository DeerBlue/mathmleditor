Communication between MathML Editor and a server

How to configure the Editor to take the mathML from a server ?

	Put in parameter FlashVars the value mathMLurl

		<param name="FlashVars" value="mathMLurl=http://yourServer/getMathML.jsp"/>

How to configure the Editor to save the mathML to a server ?

	Put in parameter FlashVars the value urlForSaveButton

		<param name="FlashVars" value="urlForSaveButton=http://yourServer/saveMathML.jsp"/>
