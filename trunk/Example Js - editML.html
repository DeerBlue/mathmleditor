Communication between Javascript and MathML Editor

To load MathML from Javascript. Click here.

To get the mathML from the Editor, click on "Save MathML" button, and the javascript method "saveMathML" would be called

How to send the mathML string to Editor ?

	function loadMathML() {
		var mathMLstring = "<mrow><mtext>cos x</mtext></mrow>";
		document.getElementById("editML").setVariable( "mathMLFromJs", mathMLstring);

	}

How to configure the Editor to call a javascript method ?

	1. Create javascript method saveMathML

		function saveMathML(mathML) {
				alert(mathML);
		}

	2. Put the parameter FlashVars for your flash object and the Editor will call the method saveMathML

		<param name="FlashVars" value="methodForSaveButton=saveMathML"/>
